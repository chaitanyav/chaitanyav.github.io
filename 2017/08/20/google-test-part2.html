Google Test - Part 2

Assertions

Google Test supports a two kinds of assertions

  • ASSERT_*
  • EXPECT_*

Assertions of type ASSERT_* are fatal i.e. failure of a ASSERT_* causes the test to fail and will prevent the assertions following it from runing whereas assertions of type EXPECT_* are non-fatal i.e. failure of a EXPECT_* will not prevent the assertions following it from running.

For a complete list of supported assertions, Please see Google Test Primer
