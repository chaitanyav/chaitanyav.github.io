Google Test - Part 1

Dissecting our first test program using Google Test (gtest)

  • The test program starts by including the header file “gtest/gtest.h”.
  • The ::testing::InitGoogleTest(&argc, argrv) call parses the commandline gtest flags to the test program and removes all unrecognised flags. It should be called before the call to macro RUN_ALL_TESTS()
  • RUN_ALL_TESTS() macro call runs all the test in the given program.
  • Each test is defined in the TEST(test_case_name, test_name) macro block. For e.g.

EXPECT_EQ(val1, val2) is a assertion macro which tests for val1 == val2
