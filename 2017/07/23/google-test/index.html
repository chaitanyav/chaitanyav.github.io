Google Test & Google Mock Installation

Download the latest version of google test from here. Type the following commands in the terminal after cd into the directory where the tarball was downloaded to.

tar -xvzf googletest-release-1.8.0.tar.gz
cd googletest-release-1.8.0
cd googletest
cmake .
make
sudo make install
cd ..
cd googlemock
cmake .
make
sudo make install

NOTE: Make sure /usr/local/lib is added to your LD_LIBRARY_PATH environment variable before using the frameworks. To set the LD_LIBRARY_PATH environment variable

~/Downloads/googletest-release-1.8.0/googlemock ᐅ export LD_LIBRARY_PATH=/usr/local/lib:$LD_LIBRARY_PATH

~/Downloads/googletest-release-1.8.0 ᐅ echo $LD_LIBRARY_PATH
/usr/local/cuda/lib:/usr/local/lib:

You may want to append/add /usr/local/lib to the LD_LIBRARY_PATH environment variable in your .bashrc or .zshrc
