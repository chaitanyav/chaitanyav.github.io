Google Test - Part 3

Custom Assertion messages

Consider the following test for a function to compute the factorial of a given integer. If you notice carefully i have introduced a bug in the factorial function. And in the test i have two EXPECT_TRUE assertions which basically assert that the factorial of 0 is 1. But the second assertion displays the custom message instead of the assert condition. This can be done with the help of AssertionResult object which can be a AssertionSuccess() or AssertionFailure().
