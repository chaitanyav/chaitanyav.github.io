SecureRandom ruby

SecureRandom module in Ruby can be used to generate random data (bytes, numbers, strings). It uses OpenSSL internally (see source code for more details).

irb(main):001:0> require 'securerandom'
=> true
irb(main):002:0> SecureRandom.class
=> Module

Random numbers:

To generate random integers less than n (n > 0)

irb(main):004:0> 10.times{p SecureRandom.random_number(100)}
43
85
60
38

11

33
17
=> 10

To generate random floating point numbers (specify n <= 0)

irb(main):007:0> 10.times{p SecureRandom.random_number(0)}

Random bytes:

To generate random n bytes

irb(main):008:0> SecureRandom.random_bytes(15)
=> "\eT\xCC\x97\xA5\x18\xFF\xEA\x9BT\xB3\xA7\xB0\x0F\x85"

for hex bytes

irb(main):011:0> SecureRandom.hex(20)
=> "88adbb214a8364c3ebf81e2203c7218ee3aaf59e"

for base64 encoded bytes

irb(main):014:0> SecureRandom.base64(20)
=> "gPHf3U0npHmSXJTT+Xz/rn0C8Xg="

for url safe base64 encoded bytes

irb(main):018:0> SecureRandom.urlsafe_base64(20)
=> "a9xtB405QiSlIDNiX9AhSz4jYq8"
